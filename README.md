<div align="center">

> *"Security is not a product, but a process."*

</div>

I'm the **founder and president** of **UCD-CYBERFORCE**, a student-led cybersecurity club at UCD.

We focus on:

| Area | Details |
|------|---------|
| 🐧 **Linux** | Security-focused Linux usage and sysadmin skills |
